# Simple Weather Forecast Website App

This website was built as a personal project using HTML, CSS and Vanilla JavaScript.
It displays the current weather details as well as the weather forecast of the next 7 days.

## How it works
1. The website first uses OpenMeteo's Geocoding API to fetch the latitude and longitude of the location name input by the user.
2. Then it uses OpenMeteo's Weather Forecast API to fetch the weather forecast data based on the location's latitude and longitude values.
