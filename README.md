# Simple Weather Forecast Website App

This website was built as a personal learning project using HTML, CSS and Vanilla JavaScript.<br>
It displays the current weather details as well as the weather forecast of the next 7 days.

## How it works
First, the website first uses [OpenMeteo's Geocoding API](https://open-meteo.com/en/docs/geocoding-api) to fetch the latitude and longitude of the location name input by the user.<br>
Then it uses [OpenMeteo's Weather Forecast API](https://open-meteo.com/) to fetch the weather forecast data based on the location's latitude and longitude values.

Live Demo: https://bhavishteeluck.github.io/simple_weather_forecast_website_app/
